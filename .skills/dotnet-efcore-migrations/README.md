# dotnet-efcore-migrations skill

Skill para orientar o Codex no VS Code a substituir `EnsureCreatedAsync()` por EF Core Migrations em projetos .NET 8 com Clean Architecture.

## Estrutura

```txt
.skills/
  dotnet-efcore-migrations/
    SKILL.md
```

## Como usar

Copie a pasta `.skills/dotnet-efcore-migrations` para a raiz do seu repositório.

Depois, no Codex dentro do VS Code, use:

```txt
Use as instruções do AGENTS.md, da skill .skills/dotnet-8-auth-microservice/SKILL.md e da skill .skills/dotnet-efcore-migrations/SKILL.md.

Substitua EnsureCreatedAsync por EF Core Migrations neste projeto.
```
